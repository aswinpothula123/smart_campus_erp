# Smart Campus ERP

A production-ready Smart Campus ERP application built with Flutter and Firebase for academic administration, student management, attendance, fees, and reporting.

## Features
- Dashboard and role-ready navigation
- Student directory, profile, and ID card workflows
- Teacher management and ID card generation
- Attendance record creation and review
- Course and fee management screens
- Analytics, reports, notifications, and settings
- Firebase-backed services for students, teachers, courses, fees, and auth

## Tech Stack
- Flutter
- Firebase Core, Auth, Firestore, Storage
- Material 3 UI

## Getting Started
1. Install Flutter.
2. Run `flutter pub get`.
3. Configure Firebase for your project.
4. Run `flutter run -d chrome`.
